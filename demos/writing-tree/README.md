# 写作树 (Writing Tree)

HWP 写作辅助工具 Demo - 帮助用户从主题出发，构建文章结构，形成树形大纲。

## 功能特性

- 🎯 **主题输入** - 输入文章主题，快速开始创作
- 🤖 **AI 生成大纲** - 自动生成多级树形章节大纲
- 🌳 **树形结构** - 可视化展示文章层级结构
- ✏️ **交互编辑** - 支持章节的展开/收起/编辑
- 🔄 **拖拽排序** - 拖拽调整章节顺序和层级
- 💾 **导出功能** - 支持导出为 Markdown 格式

## 技术栈

- React 18 + TypeScript
- Vite
- React DnD (拖拽功能)
- CSS3

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用说明

1. 在首页输入文章主题，或选择快速主题
2. 点击"生成大纲"，AI 将自动生成章节结构
3. 点击章节标题可以编辑内容和标题
4. 点击 ▼/▶ 可以展开或收起子章节
5. 拖拽章节可以调整顺序或层级
6. 点击 + 添加子章节，点击 × 删除章节
7. 使用工具栏可以全部展开/收起，或导出 Markdown
